##  Приклади

###  Приклад 1
$$
\begin{cases}
\dot{x} = -y,\\
\dot{y} = x - 3x^2
\end{cases}
$$

**Точки рівноваги:** $(0,0)$, $\left(\tfrac{1}{3},0\right)$  
**Типи:** центр, сідло
